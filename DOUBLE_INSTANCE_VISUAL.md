## Why This Happened

The LaunchAgent plist contains:
```xml
<key>RunAtLoad</key>
<true/>
```

This means: **"Start the app immediately when you load this LaunchAgent"**

So when the installer runs:
```bash
launchctl load "$LAUNCH_AGENTS_DIR/$PLIST_NAME"
```

The app already starts! No need for:
```bash
python3 "$INSTALL_DIR/$SCRIPT_NAME" &
```
